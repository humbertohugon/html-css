Principais formatações:

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (Não semântica)

Nesta frase, tenho um termo em destaque usando a tag STRONG (Semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (Não Semântica)

Nesta frase, temos um termo em ênfase usando a tag EM (Semântica)

Texto Marcado

Criando um texto marcado usando a tag MARK

Texto grande e pequeno

A criar um texto grande e outro texto pequeno. Usei as tag BIG e SMALL

Texto deletado

Marcar um texto como excluído para indicar que elee deve ser lido, mas não considerado. Usando a tag DEL

Texto Inserido

Marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usando a tag INS(Semântica)

Existe também o sublinhado usando a tag U (Não Semântica)

Texto Sobrescrito

Para inserir coisas do tipo x2 + 10. Usando a tag SUP

Para inserir coisas do tipo H20. Usando a tag SUB
